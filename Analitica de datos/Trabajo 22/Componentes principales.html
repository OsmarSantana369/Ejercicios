Transformaciones del conjunto de datos

Son algoritmos que crean una nueva representacin de los datos que podra ser ms fcil de entender en comparacin con la representacin original de los datos. Algunas de las aplicaciones de estas tranformaciones son:

  • Reduccin de la dimensionalidad: toma una representacin de alta dimensin de los datos, que consta de muchas caractersticas, y encuentra una nueva forma de representar estos datos que resume las caractersticas esenciales con menos caractersticas.
  • Encontrar las partes que componen los datos.

Reduccin de dimensionalidad

Las motivaciones ms comunes para la transformacin de datos mediante el aprendizaje no supervisado son la visualizacin, la compresin de los datos y la bsqueda de una representacin que sea ms informativa para su posterior procesamiento.

En un algoritmo, es posible que cuanta ms (y mejor) informacin se tenga, se pueda obtener una prediccin ms acertada. Pero tambin empezaremos a notar que la ejecucin de nuestro algoritmo seleccionado empezar a tomar ms y ms tiempo y recursos. Es posible que algunas de las variables sean menos importantes y no aporten demasiado valor a la prediccin.

No sera mejor tomar menos variables, pero ms valiosas?

Al quitar variables estaramos haciendo Reduccin de Dimensiones. Para reducir las dimensiones podemos hacer dos cosas:

  • Eliminar por completo dimensiones.
  • Extraccin de Caractersticas.

Se intentar mantener todas las variables posibles, pero prescindiremos de las menos importantes. Como en la extraccin de caractersticas, se tienen las variables ordenadas de mejor a peor predictora'' ya sabemos cuales sern las ms y menos valiosas.

A diferencia de la eliminacin directa de una caracterstica vieja'', nuestras nuevas variables son combinaciones de todas las variables originales, aunque eliminemos algunas, estaremos manteniendo la informacin til de todas las variables iniciales.

Uno de los algoritmos ms simples y ms utilizados para todo esto es el Anlisis de Componentes Principales (PCA).

Anlisis de Componentes Principales (PCA)

El Anlisis de Componentes Principales (PCA) es una tcnica estadstica usada para reducir la dimensionalidad de un conjunto de datos, conservando la mayor cantidad de informacin posible. Se basa en encontrar nuevas variables (componentes principales) que sean combinaciones lineales de las variables originales y que estn incorrelacionadas entre s (ortogonales).

Por qu reducir dimensiones?

  • Simplifica modelos complejos.
  • Facilita visualizacin de datos multivariados.
  • Reduce ruido y mejora desempeo en modelos de Machine Learning.

Proceso del PCA

En resumen lo que hace el algoritmo es:

  • Estandarizar los datos de entrada.
  • Obtener los valores y vectores propios de la matriz de covarianzas.
  • Ordenar los valores propios de mayor a menor y elegir los vectores propios que se correspondan con los valores propios ms grandes.
  • Construir la matriz de proyeccin con los vectores propios seleccionados.
  • Transformar el dataset original va la matriz de proyeccin para obtener las nuevas caractersticas.

Resultados del PCA

Con PCA se puede obtener:

  • una medida de como cada variable se asocia con las otras (matriz de covarianza)
  • La direccin en las que nuestros datos estn dispersos (vectores propios)
  • La relativa importancia de esas distintas direcciones (valores propios)
